# Devcamper API

> Backend API para aplicação DevCamper.

## Usage

Renomear "config/config.ebv.env" para "config/config.env" e altere os valores e configurações como achar melhor.

## Instalar Dependências

'''
npm install
'''

# Run App

'''

# Run in dev mode

yarn run dev

# Run in prod mode

nom start
'''
